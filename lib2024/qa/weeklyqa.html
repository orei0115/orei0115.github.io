  1. Question 1: What are the three components of Web?

    Answer 1: HTML, CSS, JavaScript

  2. Question 2: Define version control and give one example.

    Answer 2: Version control is a system that records changes to a file or set of files over time so that you can recall specific versions later. An example of version control software is GitHub.

  3. Question 3: What tag is used to create the title of a web page? Give an example.

    Answer 3: The tag < title > is used to create a title. An example is the one I used in this html document.

  4. Question 4: What tag is used to create a paragraph in html? Give an example.

    Answer 4: The tag < p > is used to create a paragraph. An example in this document seperates each question and answer set.

  5. Question 5: Identify the odd one from these tags < span > , < p > , < a > , < i > and state the reason?

    Answer 5: The < i > tag for italics is the odd one as it is the only tag that affects the style of the page.

  6. Question 6: What are the tags required to create a navigation list in HTML?

    Answer 6: The tags used for navigation in html are < nav > and < href >

  7. Question 7: What element is used to create images? Name its attributes.

    Answer 7: The tag < img > is used to add an image. It needs an src attribute < img src > and can be used to put the URL in quotes, or upload it to sublime text. The < alt > will let you define the size and make the image able to be describable for the visually impaired.

  8. Question 8: Identify the odd one from these tags: < table > , < tr > , < b > , < td > and state the reason.

    Answer 8: The < b > tag is the odd one out because it bolds the text whereas the other three tags are for adding a table, adding table data, and defines the table rows.
